# Homebrew Tap for kulesh's Tools

This is a [Homebrew](https://brew.sh) tap for installing kulesh's command-line tools.

## Installation

To install tools from this tap:

```bash
# Tap this repository
brew tap kulesh/tap

# Install tools
brew install tc
```

Or install directly without tapping:

```bash
brew install kulesh/tap/tc
```

## Available Tools

- **tc** - Token counter for LLM development (similar to Unix `wc`)

## About

Formulae in this tap are automatically generated and updated by [cargo-dist](https://github.com/axodotdev/cargo-dist).
